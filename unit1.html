How to add a stylesheet to an HTML file

There are three different ways to incorporate styles to an HTML file

  • Inline styles

    • styles applied within the HTML element
    • Example:
    • <p style=“font-size:20px;”></p>

  • Embedded Style Sheets

    • Style are applied within the HTML file, in the head section of the file
    • Example:
    • <head>
        <style type=“text/css”>
          { /* stylesheet statemets go here */ }
        </style>
      </head>

  • External Style Sheets

    • styles are in a separate .css file and referenced in the head of the HTML file
    • Example:
    • <head>
        <link rel=“stylesheet” href=“style.css”/>
      </head>

    • Mime Types
