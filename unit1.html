HTML

  • Hypertext Markup Language (HTML) is used to create the structure of a webpage.
  • Example:
  • <!DOCTYPE html>
    <head>
       <title>Sample webpage</title>
       <meta charset="utf-8">
    </head>
    <body>
       <h1>Hello World!</h1>
    </body>

Basic HTML elements

  • The structure of the HTML file, and hence the webpage, is defined by the HTML elements. Below are some of the most useful HTML elements and their functionalities

  • head

    • the head element defines some of the properties of the webpage including the title, meta information as well as stylesheets and scripts

  • body

    • the content of the web page is contained in the body element.

  • div

    • groups elements and page contents in a section
    • Example:
    • <div>
         <p>Here I will talk about my favourite movies... </p>
      </div>

      <div>
         <p>Here I will talk about my favourite food... </p>
      </div>

  • p

    • p is used for paragraphs
    • From the example above, th element p was used within the div

  • li

    • The element li enumerates list items in bullet form or numbered format
    • It's also usually nested within <ul> or <ol> tags

  • ul

    • <ul> stands for unordered list. Meaning the list below doesn't require order and by default, will show a bullet point list
    • Example:
    • <h3>list of European countries:</h3>
      <ul>
         <li>France</li>
         <li>Italy</li>
         <li>Spain</li>
         <li>Netherlands</li>
      </ul>

    • <ol> stands for ordered list. This is used instead of using ul onlists that requires numbering.

    • This element is a way to add hyperlinks within the web page.

    • <p>Click <a href="www.example.com">here</a> to go to the example page.</p>
    • clicking the hyperlink goes to the link specified: example.com
    • href is the attribute used with the a to specify the link that's attached.

CSS: Cascading Style Sheets

  • CSS is used to “style” web pages
  • it is used to specify presentation of elements separately from the structure of the document

Validating CSS

  • you can validate your CSS file by going to jigsaw.w3.org/css-validator/

How to add a stylesheet to an HTML file

There are three different ways to incorporate styles to an HTML file

  • Inline styles

    • styles applied within the HTML element
    • Example:
    • <p style=“font-size:20px;”></p>

  • Embedded Style Sheets

    • Style are applied within the HTML file, in the head section of the file
    • Example:
    • <head>
        <style type=“text/css”>
          { /* stylesheet statemets go here */ }
        </style>
      </head>

  • External Style Sheets

    • styles are in a separate .css file and referenced in the head of the HTML file
    • Example:
    • <head>
        <link rel=“stylesheet” href=“style.css”/>
      </head>
