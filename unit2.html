Displaying a line of text in a web page using JavaScript

Similar with CSS in the previous unit, there are also multiple ways of incorporating a JS script to a webpage:

  • Embedded scripts

    • like the way we embed style sheets in an HTML, we can do a similar thing to JavaScript by adding in a script in between the <script> tag (HTML immediately knows that what follows is a script). The difference is that an embedded style scheet is strictly enclosed in the <head/> tag, and a script can go anywhere in the code.
    • Example:
    • <head>
         <meta-charset="utf-8">
         <title>Sample Embedded script</title>
         <script type="text/javascript">
           document.writeln("<h1/>Hello World</h1>");
         </script>
      </head>

  • External Scripts

    • Again, similar to CSS, we can add external scripts by having it as a source (src) to the script tag
    • A JS script can be saved as a .js file
    • Example:
    • <head>
         <meta-charset="utf-8">
         <title>Sample Embedded script</title>
         <script src="app.js" type="text/javascript"> </script>
      </head>

Control Statements in JavaScript

  • If-else statements

    • if-else statements are used if a section of the code can only be executed if certain conditions are met
    • Example:
    • var x = 5;

      if (x < 3) {

        console.log("apples");
      }

      else {
        console.log("oranges");
      }

      // prints "oranges" to the console

  • Switch statements

    • switch statements are useful if you have multiple different conditions to use. In this case, switch statements are more readable than using if-else statements.
    • Example:
    • var animal = dog;

      switch (animal) {

        case "cat":
          console.log("meow!");

        case "cow":
          console.log("moo!");

        case "dog":
          console.log("woof!");

        case "horse":
          console.log("neigh!");

        default:
          console.log("");

      }

      // prints "woof!" to the console
    • Note: a default case should be use if in case any of the given cases doesn't work.

  • While loop:

    • a while loop is used if a statement has to be executed multiple times while the condition is satisfied
    • Example:
    • var i = 1;
      var str = "";

      while (i <= 5) {
        str += i + " ";
        i = i + 1;
      }

      console.log(str);

      // prints "1 2 3 4 5 " to the console

  • For loop:

    • a for loop is used if a statement has to be executed given a certain amount of times.
    • Example:
    • var str = "";

      fot (var i = 1; i <= 5; i++) {
        str += i + " ";
      }

      console.log(str);

      // prints "1 2 3 4 5 " to the console
    • Note: Use for loop instead of while loop if you're certain about number of times it's goonna be repeated

HTML Canvas

  • The canval element provides a JavaScript Application Programming Interface (API) for drawing 2-D images

Canvas Coordinate System

  • The first thing you need to know before starting with canvas is knowing the coordinate system. The origin is at the upper-left side of the screen.

Drawing attributes

  • Below is an example on how to use the canvas element on HTML as well as using some drawing attributes to draw a rectangle
  • Example:
  • <head>
       <meta charset="utf-8">
       <title>Sample Embedded script</title>
    </head>
    <body>
       <canvas id="drawRectangle" width="300" height="100"
         style="border: 1px solid black;">
       </canvas>
       <script type="text/javascript">
         var canvas = document.getElementById("drawRectangle");
         var context = canvas.getContext("2d");
         context.fillStyle = "yellow";
         context.fillRect(5, 10, 200, 75);
         context.strokeStyle = "royalblue";
         context.lineWidth = 6;
         context.strokeRect(4, 9, 201, 76);
       </script>
    </body>
  • Output:
  • The width and the height of the canvas element should be specified. This will be the dimension of the drawing. It should also have an id so that the script will be able to identify it.
  • var context = canvas.getContext("2d");
    The getContext() method gets the context object. It represents the 2d rendering surface. The object also as the methods for drawing: fillStyle(), fillRect(), strokeStyle(), lineWidth(), strokeRect() are some of the examples (more examples of other methods on the next section)s. It also does font manipulation, color manipulation and other methods related to graphics.

Drawing Complex Shapes with Paths

  • Canvas can draw paths by specifying the coordinates where it begins, and the points where it goes to next.
  • Example:
  • <body>
       <canvas id="drawPath" width="400" height="200"
         style="border: 1px solid black;">
       </canvas>
       <script type="text/javascript">
         var canvas = document.getElementById("drawPath");
         var context = canvas.getContext("2d");
         context.beginPath();
         context.moveTo(10,30);
         context.lineTo(390, 60);
         context.lineTo(40, 90);
         context.lineTo(360, 120);
         context.lineTo(70, 150);
         context.lineTo(330, 180);
         context.lineWidth = 8;
         context.lineJoin = "bevel";
         context.lineCap = "butt";
         context.strokeStyle = "purple";
         context.stroke();
       </script>
    </body>
  • Output:
  • Here, we introduced more methods in the canvas context object: beginPath() which starts the drawing, and stroke() to finish it. lineTo() tells wo what coordinates the path should go to next. lineJoin() and lineCap() specifies what the joints and the ends respectivelly will look like. strokeStyle() specifies the color of the stroke.

Drawing Arcs

  • Canvas can not only lines, it also has a way to draw arcs and curved lines.
  • Example:
  • <body>
       <canvas id="drawPath" width="225" height="110"
         style="border: 1px solid black;">
       </canvas>
       <script type="text/javascript">
         var canvas = document.getElementById("drawArcs");
         var context = canvas.getContext("2d");
         context.beginPath();
         context.arc(35, 50, 30, 3*Math.PI/2, Math.PI, false);
         context.strokeStyle = "purple";
         context.stroke();

         context.beginPath();
         context.arc(110, 50, 30, 0, Math.PI/2, true);
         context.strokeStyle = "purple";
         context.stroke();
       </script>
    </body>
  • Output:
  • the arc() method has the following parameters: first two is the x and y coordinate of the center of the arc. the third parameter is the radius of the arc. The fourth and fifth parameter is the starting and ending angle respectivelly in radians. The image below shows where the angles are.
  • The sixth parameter is optional. It tells the direction of the arc stroke. The value 'true' corresponds to counterclockwise, and 'false' corresponds to clockwise.
